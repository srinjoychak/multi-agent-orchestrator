# Review

Use this skill after a meaningful change and before merge.

## Review focus

- correctness
- security
- regressions
- missing tests
- maintainability

## Output format

- Scope
- Findings by severity
- Suggested fix for each finding
- Verdict

## Rule

- Do not skip review because the change looks small.
