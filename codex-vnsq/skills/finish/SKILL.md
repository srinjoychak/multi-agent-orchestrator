# Finish

Use this skill when the branch is ready to close out.

## Workflow

1. Run the project's verification commands.
2. Confirm the base branch.
3. Present the completion options.
4. Execute the chosen option.
5. Clean up worktrees when needed.

## Options

- Merge locally
- Push and create a PR
- Keep the branch
- Discard the work
